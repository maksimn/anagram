#### Дано

Текстовый файл, в каждой строке которого содержится одно слово.

#### Задача 1

Написать на C# консольное приложение, которое должно найти все анаграммы (слова, отличающиеся только порядком букв) в заданном файле. 
Результат работы приложения - текстовый файл, каждая строчка которого содержит одну группу анаграмм исходного файла, например: первая строка - "ток, кот, кто", вторая строка - "вертикаль, кильватер", третья - "колун, уклон" и т.д.

#### Задача 2

Найти все анаграммы при условии, что файл большой (например, 1 ТБ), а памяти немного (приложение не должно съесть больше 1 ГБ). Задача имеет множество возможных решений. Нужно найти решение, выполняющеее поставленную задачу как можно быстрее.
